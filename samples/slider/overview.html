The wijslider widget simplifies user input by providing a rich visual representation of numeric changes. The wijslider is a sliding selector that provides a simple, familiar way for end-users to choose a single value or a range of values in a predefined range at run time.

The wijslider can be used either horizontally or vertically. This sample shows how to set the wijslider in either way, by using the orientation option.
