Create a professional and polished website with the help of the wijsplitter widget. wijsplitter is a container widget featuring a movable and collapsible bar that divides a container's display area into two resizable panels. Splitters are able to be nested indefinitely, providing you infinite possibilities in UI design. The samples in this section will help to get you started with the wijsplitter widget.
